# Estudio de Percepción: Smartphones Gama Alta 2025

Este proyecto consiste en un ejercicio estadístico desarrollado en Python utilizando Google Colab.

## Objetivo

Recolectar votos de usuarios para identificar cuál es la marca de smartphone de gama alta más valorada en 2025.

## Funcionalidades

* Registro de votos por marca
* Registro de edad del participante
* Generación de tabla de resultados
* Cálculo de:

  * número de votos
  * edad promedio
  * edad mínima y máxima
* Exportación de datos a CSV

## Tecnologías utilizadas

* Python
* Pandas
* ipywidgets
* Google Colab

## Archivo principal

`AlgoritmoRecomendacion.ipynb`

## Resultado

El sistema genera una tabla con el ranking de preferencia de marcas premium:

* Apple
* Samsung
* Huawei
* Xiaomi
* Google
